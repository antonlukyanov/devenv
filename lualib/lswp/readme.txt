lswp.gpf_off()
  Для текущего процесса и всех порожденных процессов отключает
  вывод диалога, сообщающего о General Protection Fault.

lswp.set_background()
  Устанавливает фоновый приоритет для текущего процесса.

lswp.sleep(ms)
  Усыпляет текущий поток на ms миллисекунд.

lswp.kill(pid)
  Убивает процесс с заданным идентификатором.

lswp.ps()
  Возвращает список процессов в виде таблицы следующего вида:
  {
    { pid = xxx, name = "yyy", path = "zzz" },
    ...
  }
