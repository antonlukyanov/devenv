lswp.gpf_off()
  Для текущего процесса и всех порожденных процессов отключает
  вывод диалога, сообщающего о General Protection Fault.

lswp.set_priority(p)
  Устанавливает приоритет p для текущего процесса.
  Допустимые значения приоритета:
    -2 фоновый
    -1 пониженный
     0 нормальный
     1 повышенный
     2 высокий

lswp.sleep(ms)
  Усыпляет текущий поток на ms миллисекунд.

lswp.kill(pid)
  Убивает процесс с заданным идентификатором.

lswp.ps()
  Возвращает список процессов в виде таблицы следующего вида:
  {
    { pid = xxx, name = "yyy", path = "zzz" },
    ...
  }

lswp.run(cmdl[, p])
  Запускает процесс, используя параметры из таблицы p.
  Значения полей таблицы:
         pwd - рабочая директория
               (при отсутствии используется рабочая директория процесса-предка).
    priority - приоритет приложения
               (при отсутствии наследуется приоритет процесса-предка).
        wait - ожидать завершения процесса 
               (true/false, при отсутствии - true)
      window - стартовое состояние окна 
               ("normal", "min", "hide", при отсутствии - normal).
